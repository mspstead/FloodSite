    About

    This website resource was produced for a final year research project for the School of Computing, University of Leeds.
    This website is a proof of concept project, potential data sources related to flooding have been visualised on the website.
    This page uses Environment Agency real-time data API (Beta) to gather flood warnings.

    Map

    The map page plots geo-located photographs of flooding in West Yorkshire sourced from Flickr and Instagram.
    There are also geo-tagged tweets plotted to this map related to the floods which affected West Yorkshire starting on the 26/12/2015

    Timeline

    The timeline page displays a graph of river levels of several river level gauges situated in West Yorkshire.
    The timeline page also displays photographs and tweets on a descending timeline.
    Both the river level data and timeline can be filtered based on different flood event periods.
    Each photo displayed on the timeline has a score. These scores are decided by voting on the photos using the up and down buttons.
