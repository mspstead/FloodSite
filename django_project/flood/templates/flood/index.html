    About

    This website resource was produced as part of a research project for the University of Leeds.
    This website uses Environment Agency flood and river level data from the real-time data API (Beta)

    Map

    The map page consists of geo-located photographs sourced from Flickr and Instagram.
    There are also geo-tagged tweets plotted to this map relating to the flooding on the 26/12/2015

    Timeline

    The timeline page provides a breakdown of all the flood data (photos, rain-levels and tweets)
